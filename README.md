## What's included?
 * ClassLoader
 * Routing
 * Sessions
 * Config
 * Tokens
 * Templating

### Requirements
 
 - PHP =< 7.0
 - Rewrite Module (nginix/apache2)
 - Composer
 

### Installation

Framework Core
```sh
$ composer require dframe/dframe
```

Demo skeleton project 
```sh
$ composer create-project dframe/dframe-demo project_name
```
